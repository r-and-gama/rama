1.1 Desiging experiments

We can make an experiment object either by reading an experiment from a GAML file with the load_experiment() funtion or by building it from scratch with the experiment() constructor.

1.1.1 Loading an experiment from a GAML file

Let’s use the following gaml file:

gaml_file <- system.file("examples", "sir.gaml", package = "rama")

Loading the experiment sir from this file. Note that we also indicate the names of the folder where we wish to save the simulations outputs. If not specified, the name of the .gaml is used by default:

exp1 <- load_experiment("sir", gaml_file, "sir")
#> Simulations results will be saved in "/Users/haihb/Documents/Work/rama/vignettes/sir_1".
#> 
#> Loading experiment "sir" from file "sir.gaml"...
#>

which gives:

exp1
#> Experiment with 1 simulation of 5 parameters and 3 observed variables
#> experiment name:     sir 
#> input gaml file:     /Users/haihb/Documents/Work/rama/inst/examples/sir.gaml 
#> output directory:    /Users/haihb/Documents/Work/rama/vignettes/sir_1 
#> model parameters:    p_S0, p_I0, p_R0, p_beta, p_gamma 
#> observed variables:  r_S, r_I, r_R 
#> Experiment overview:
#>   p_S0 p_I0 p_R0 p_beta p_gamma r_S r_I r_R tmax seed
#> 1  999    1    0    1.5    0.15   1   1   1 1000    1

An experiment is a simple data frame:

class(exp1)
#> [1] "experiment" "data.frame"

This implies that we will be able to use all the data frame methods to manipulate experiments.

1.1.2 Building an experiment from scratch

Alternatively, an experiment can be built from scratch with the experiment constructor:

df <- data.frame("S0" = 999, "I0" = 1, "R0" = 0, "beta" = 1.5,
                 "gama" = 0.15, "S" = 1, "I" = 1, "R" = 1,
                 "a" = 1000, "b" = 1)
exp2 <- experiment(df, 
           parameters = c("S0", "I0", "R0", "beta", "gama"),
           obsrates = c("S", "I", "R"), tmax = "a", seed = "b",
           experiment = "sir", model = gaml_file)
#> Using default directory name "sir" in current directory "/Users/haihb/Documents/Work/rama/vignettes".
#> 
#> Simulations results will be saved in "/Users/haihb/Documents/Work/rama/vignettes/sir_2".
#>

Which gives the same thing:

exp2
#> Experiment with 1 simulation of 5 parameters and 3 observed variables
#> experiment name:     sir 
#> input gaml file:     /Users/haihb/Documents/Work/rama/inst/examples/sir.gaml 
#> output directory:    /Users/haihb/Documents/Work/rama/vignettes/sir_2 
#> model parameters:    p_S0, p_I0, p_R0, p_beta, p_gama 
#> observed variables:  r_S, r_I, r_R 
#> Experiment overview:
#>   p_S0 p_I0 p_R0 p_beta p_gama r_S r_I r_R tmax seed
#> 1  999    1    0    1.5   0.15   1   1   1 1000    1

Note that if you enter values of inequal lengths, it will automatically complete:

df <- data.frame("S0" = c(999, 990), "I0" = c(1, 2), "R0" = 0, "beta" = 1.5,
                 "gama" = 5, "S" = 1, "I" = 1, "R" = 1,
                 "a" = 1000, "b" = 1)
exp3 <- experiment(df, parameters = c(1:5),
           obsrates = c(6:8), tmax = "a", seed = "b",
           experiment = "sir", model = gaml_file)

Which gives:

exp3
#> [1] "Unknown class"

Note finally, that we can make smart use of the expand.grid() function of the base package to efficiently generate a complete experimental design:

df <- expand.grid(S0 = c(900, 950, 999),
               I0 = c(100, 50, 1),
               R0 = 0,
               beta = 1.5,
               gamma = .15,
               S = 1,
               I = 1,
               R = 1,
               tmax = 1000,
               seed = 1)

exp4 <- experiment(df, parameters = c(1:5),
           obsrates = c(6:8), tmax = "tmax", seed = "seed",
           experiment = "sir", model = gaml_file)

Which gives:

exp4
#> [1] "Unknown class"

We’ll see more about that in the next sections.

1.1.3 Building an experiment from a data frame

df <- data.frame(
  S0 = c(700, 800, 900, 950, 999),
  R0 = 0,
  beta = 1.5,
  gamma = .15,
  S = 1,
  I = 1,
  tmax = 2500,
  seed = sample(5)

df$I0 <- 1000 - df$S0

1.3 Running experiments

Then, we run the experiment by one command run_experiment, in which we specify the experiment and optionally number of cores, output folder for the results of this experiment, parameters file in xml format.

output <- run_experiment(exp1)
